Details

These binary are built with command showing below.

Build Command:

curl -L -o "nodejs-$VERSION.zip" https://github.com/nodejs/node/archive/refs/tags/$VERSION.zip
unzip "nodejs-$VERSION.zip"
cd "node-${VERSION:1}"
./configure --prefix=/ --dest-cpu=arm --download=all --with-intl=full-icu
make -j8 binary V= DESTCPU=arm64 ARCH=arm64 DISTTYPE=release
