Node.JS 14 Apple Silicon (arm64) Unofficial Build

This site provide unofficial apple silicon (arm64) build.

These files the command shown below to build.

Build Command:

curl -L -o "nodejs-$VERSION.zip" https://github.com/nodejs/node/archive/refs/tags/$VERSION
unzip nodejs-*
cd nodejs-*
./configure --prefix=/ --dest-cpu=arm --download=all --with-intl=full-icu
make -j8 binary V= DESTCPU=arm64 ARCH=arm64 DISTTYPE=release
